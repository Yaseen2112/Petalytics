# 🌸 Iris Flower Classifier

This project predicts the species of an Iris flower based on its physical measurements using a trained machine learning model. It features a user-friendly GUI for input and displays the predicted flower type.

---

## Project Overview

The Iris dataset is a classic dataset in machine learning and pattern recognition. It contains measurements of sepal length, sepal width, petal length, and petal width from three different species of Iris flowers:
- Iris-setosa
- Iris-versicolor
- Iris-virginica

Using these measurements as input features, this project trains a Random Forest classifier to predict the species of an Iris flower.

---

## Files in the Project

- `iris.csv`: Dataset containing the flower measurements and species labels.
- `train_model.py`: Script to train the Random Forest model on the dataset and save the model as a pickle file.
- `model.pkl`: Pickled trained model generated by `train_model.py`.
- `main.py`: Python script featuring a GUI built with Tkinter that allows users to input flower measurements and receive species prediction.
- `requirements.txt`: Python dependencies required to run the project.

---

## Setup and Usage

### 1. Install Dependencies

Install all required packages using pip:

pip install -r requirements.txt

### 2. Train the Model

Run the training script to generate the model file:

python train_model.py

This loads the dataset, trains a Random Forest classifier, and saves the trained model as `model.pkl`.

### 3. Run the Application

Launch the GUI app:

python main.py

A window will appear where you can input the measurements of an Iris flower's sepal and petal. Click the "Predict Flower Type" button to see the predicted species.

---

## Project Details

- **Model**: Random Forest Classifier from scikit-learn.
- **Framework**: Python 3 with Tkinter for GUI.
- **Data**: Iris dataset with 150 samples including flower measurements and species.

---

## How It Works

1. The user inputs four numerical values: sepal length, sepal width, petal length, and petal width.
2. The GUI validates the input and sends the data to the trained model.
3. The model predicts the species based on the input features.
4. The predicted species is displayed to the user.
